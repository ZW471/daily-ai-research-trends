# Daily AI Research Trends

A Next.js webapp that provides automated daily analysis of AI/ML research trends.

**Live site:** [https://daily-ai-research-trends.netlify.app](https://daily-ai-research-trends.netlify.app)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is deployed on [Netlify](https://www.netlify.com/) using the `@netlify/plugin-nextjs` plugin. Pushes to the `main` branch trigger automatic deployments.

- **Live URL:** [https://daily-ai-research-trends.netlify.app](https://daily-ai-research-trends.netlify.app)
- **Admin:** [https://app.netlify.com/projects/daily-ai-research-trends](https://app.netlify.com/projects/daily-ai-research-trends)
